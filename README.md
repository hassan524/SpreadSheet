# React Spreadsheet UI (Tailwind Assessment)

A spreadsheet-style task table made with **React 19**, **TypeScript**, **TailwindCSS**, and **TanStack Table v8** — includes dynamic row rendering, status chips, and icon-based headers.

## ⚙️ Tech Stack

- React 19 + TypeScript  
- Tailwind CSS 4  
- @tanstack/react-table v8  
- Bootstrap Icons  
- Simplebar (custom scrollbars)  
- Vite 7

## 🚀 Setup & Run (Single Command)

```bash
git clone https://github.com/hassan524/SpreadSheet.git && cd SpreadSheet && npm install && npm run dev
```

### 📦 Install Dependencies

```bash
npm install
```

### Build for Production

```bash
npm run build
```

### For preview

```bash
npm run start
```
